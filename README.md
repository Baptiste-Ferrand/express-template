# express-template
An template Rest Api for expressTs
For MongoDB using Mongoose as orm

Architecture 
- Routers : (instancie the endpoint and expose route)
- Controllers : (control data and validate)
- Services: (mutate data and return data from bd)

Using bcrypt for hash password


Using a docker compose for self hosted an mongodb
First copy .env.example as .env and replace the variable (u can just use the variables I give in the example)

Just tape for build mongodb
```bash
docker compose up -d
```

For starting the dev 
```bash
npm run dev
```
